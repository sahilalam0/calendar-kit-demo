# Calendar Kit Demo

Demo app for [@howljs/calendar-kit](https://github.com/howljs/react-native-calendar-kit)

## Install Dependencies

```bash
npm install
```

## Clean Prebuild

```bash
npx expo prebuild --clean
```

## Run Development Builds

iOS:

```bash
npm run ios
```

Android:

```bash
npm run android
```
